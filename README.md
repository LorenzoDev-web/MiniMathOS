# MiniMathOS
this is a mini operating system focused on mathematics, created solely by me; this system is just a hobby prototype; it has an interactive terminal with a unique language based on Basic.

to install it is necessary to compile it using "make" in your Linux terminal; with the execution of this command a .ISO file will be created; to run it you can use QEMU (by executing the command "make run"), or using a boot flash tool, like Ventoy.

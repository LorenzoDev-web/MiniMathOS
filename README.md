# MiniMathOS
English:
this is a mini operating system focused on mathematics, created solely by me; this system is just a hobby prototype; it has an interactive terminal with a unique language based on Basic.

to install it is necessary to compile it using "make" in your Linux terminal; with the execution of this command a .ISO file will be created; to run it you can use QEMU (by executing the command "make run"), or using a boot flash tool, like Ventoy.

Português-Brasil:
este é um mini sistema operacional focado em matemática, criado unicamente por mim; este sistema é apenas um protótipo de hobby; ele possui um terminal interativo com uma linguagem única baseada em Basic.

para instalá-lo é necessário compilá-lo usando "make" no seu terminal Linux; com a execução deste comando, um arquivo .ISO será criado; para executá-lo você pode usar o QEMU (executando o comando "make run"), ou usando uma ferramenta de boot em flash, como o Ventoy.
